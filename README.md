# SIGIAF - Gestión de Proyectos

Repositorio académico del proyecto `SIGIAF` (Sistema Inteligente de Gestión y Auditoría Fiscal con Inteligencia Artificial), elaborado para la asignatura Gestión de Proyectos de Software de las UTS.

## Objetivo del proyecto

SIGIAF propone un aplicativo web para apoyar procesos de auditoría fiscal mediante:

- gestión de auditorías y expedientes;
- carga y procesamiento documental;
- OCR para extracción de texto;
- clasificación documental con IA;
- detección preliminar de alertas de riesgo;
- dashboards de seguimiento;
- generación de informes preliminares;
- trazabilidad de acciones críticas.

## Estado del repositorio

Este repositorio contiene principalmente artefactos de gestión, análisis, arquitectura y pruebas del proyecto. Su estado actual corresponde a una versión académica de formulación y planeación, reforzada con artefactos de control para mejorar trazabilidad, seguimiento y calidad documental.

## Metodología

El proyecto se plantea bajo Scrum en un horizonte de 6 meses, distribuidos en 12 sprints de 2 semanas. A nivel técnico, la propuesta documentada contempla:

- frontend en React;
- backend en NestJS;
- base de datos PostgreSQL;
- OCR para extracción documental;
- módulo de IA en Python;
- despliegue de pruebas con Docker.
